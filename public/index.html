Webhook Integration Guide

Configure your MikroTik router to send script executions to the following address every 30 seconds.

http://localhost:2041/webhook/router1?key=thiskey219Kx&router=MainRouter&rx=100&tx=100&users=user1;user2;

Example MikroTik Netwatch Script:

/tool fetch url="http://<your-server-ip>:2041/webhook/router1\?key=thiskey219Kx&router=MainRouter&rx=$[interface get ether1 rx-byte]&tx=$[interface get ether1 tx-byte]&users=$[/ppp active print as-value]" keep-result=no
